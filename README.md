## 🧪 과제 1: 시스템 프로그래밍

### 학습 내용
- 표준 입력 및 출력 처리
- 문자열과 배열 사용
- 함수 호출과 스택 메모리 구조
- 입력 처리 방식에 따른 프로그램 동작 차이

### 학습 포인트
- C 프로그램의 메모리 사용 흐름 이해
- 안전한 입력 처리의 중요성 인식
- 시스템 프로그래밍 관점에서 코드 분석 능력 향상

## 📘 과제 2: 메모리 구조 및 입력 처리 분석

### 학습 내용
- 입력 데이터가 메모리에 저장되는 방식 분석
- 문자열 및 배열 처리 시 발생할 수 있는 문제 이해
- 입력 검증 로직의 필요성 학습

### 학습 포인트
- 프로그램이 비정상 동작하는 원인을 구조적으로 분석
- 메모리 관점에서 프로그램 동작 해석
- 안정적인 시스템 소프트웨어 설계의 기초 이해
